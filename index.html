FreeCodeCamp API Project

Image Search Abstraction Layer

Example usage:

https://isal.herokuapp.com/cats outputs search results for keyword "cats" in this format:

{"url":"___","title":"___","posted_by":"___","related_images":"___"}

https://isal.herokuapp.com/cats?offset=2 navigates to second page of results.

https://isal.herokuapp.com/history/latest outputs most recently submitted searches in this format:

{"term":"___","when":"___"}
